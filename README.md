# 🎫 Customer Support Ticket Management Web App

A responsive web application built using HTML, CSS, and JavaScript to manage customer support tickets efficiently.

---

## 🚀 Live Demo
🔗 https://karnatakalokesh.github.io/customer-support-ticket-app/

---

## 📌 Project Overview
This project is designed to simulate a real-time customer support system where users can create, track, and manage support tickets. It helps in understanding how customer service workflows operate in real-world applications.

---

## ✨ Features
- Create new support tickets
- Add customer name, issue description, and priority
- Track ticket status (Open / In Progress / Closed)
- Dynamic ticket display using JavaScript
- Responsive and user-friendly UI

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
